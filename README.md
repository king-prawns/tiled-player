# Tiled Player

Tiled MSE Video Player

## Installation

- Use the right version of node

```sh
nvm use
```

- Install dependencies

```sh
npm ci
```

## Sandbox

- Install dependencies

```sh
cd sandbox
npm ci
```

- Run Sandbox

```sh
npm run dev
```
